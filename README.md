# tomatox

## Clone repo

1. Clone repository

```bash
git clone https://github.com/baetmon/tomato
```
2. Change directory

```bash
cd tomato
```

## Install deps

```bash
npm install
```
## Setup Bot

```bash
npm run setup
```

## Run script

```bash
npm start
```

## Add user

You can run this step after setup bot.

- Open your telegram bot that you setup previously
- Send the auth url copied, see [video](https://www.youtube.com/shorts/S0apuiFTyuI)
- When you send the copied url or text bot will automatically add user file.
- Only admin (who setup bot initially with user id will able to add/remove user)
